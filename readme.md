# hanchor [![Build Status](https://travis-ci.org/rod/hanchor.svg?branch=master)](https://travis-ci.org/rod/hanchor)

> Generate links to your header tags


## Install

```
$ npm install --save hanchor
```


## Usage

```js
const hanchor = require('hanchor');

hanchor('unicorns');
//=> 'unicorns & rainbows'
```
